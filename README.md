# コントリビューション

## 書き方の方針
できるだけ堅い表現をさけて柔らかい表現を使いましょう。

| 堅い表現 | 柔らかい表現 | 
| ------- | ---------- |
| 行う | する |
| 異なる | 違う |

[ひらがな書きが適当な語](http://www.yamanouchi-yri.com/yrihp/techwrt-2-4s/t-2-4s03a.htm)を意識しましょう。
